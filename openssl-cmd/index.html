输出x509证书信息

openssl x509 -noout -text  -in ca.pem

验证公钥私钥是否匹配

diff -eq <(openssl x509 -pubkey -noout -in cert.crt) <(openssl rsa -pubout -in cert.key)

正常会输出

writing RSA key

验证证书CA

openssl verify -CAfile ca.pem client.pem

正常输出

client.pem: OK
